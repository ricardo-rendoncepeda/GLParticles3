ABOUT
GLParticles3 is the third and final part of an OpenGL ES particle system tutorial series, teaching developers how to make a cool and fun particle system from scratch, and integrate it into an iOS app!
In part 3 you'll use your newly developed skills to integrate particle effects into a simple 2D game.
The full tutorial is available at:
<http://www.raywenderlich.com/38384/opengl-es-particle-system-tutorial-part-3>


ACCOMPANYING FILES
The full project includes:
-Resources: Point sprite textures in three different resolutions: 16x16, 32x32, 64x64 pixels. One game sound effect.
-Xcode: A full Xcode project ready to build and run!
